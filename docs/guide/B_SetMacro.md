# B_SetMacro

## 功能说明

批量设置或重置 MedBox 预设别名与快捷键。

## 源码要点

- 输入：先选择命令模式
- 输出：写入或重置 Rhino 别名与快捷键
- 可调项：Change、重置
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：先选择命令模式。
2. 根据需要调整命令行选项：Change、重置。
3. 确认执行后，系统会写入或重置 Rhino 别名与快捷键。

## 注意事项

- 会直接写入 Rhino 的别名和快捷键设置。
