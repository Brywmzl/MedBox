# B_SelSimilar

## 功能说明

以参考物件为基准筛选相似物件并直接选中。

## 选项说明

- 输入：选择一个参考物件
- 输出：选中相似物件，并在命令行输出数量
- 可调项：无明显命令行选项。
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：选择一个参考物件。
2. 确认执行后，系统会选中相似物件，并在命令行输出数量。

## 注意事项

- 锁定物件会被排除。
