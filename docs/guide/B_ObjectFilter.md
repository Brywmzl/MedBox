# B_ObjectFilter

- 输入：在物件过滤对话框中填写参数

- 选项记忆：本次源码未见持久化选项。

1. 打开命令后，先在对话框中填写需要的参数或筛选条件。
2. 按提示准备输入：在物件过滤对话框中填写参数。
3. 确认执行后，系统会在对话框确认后执行对应处理。
