# B_UniformDirection

## 功能说明

依据参考线方向统一曲线方向或曲面法向。

## 源码要点

- 输入：按命令过滤选择物件；拾取一条方向线或参考线
- 输出：直接替换原始物件
- 可调项：无明显命令行选项。
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：按命令过滤选择物件、拾取一条方向线或参考线。
2. 确认执行后，系统会直接替换原始物件。
