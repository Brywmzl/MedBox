# B_SetSequence

## 功能说明

沿参考曲线为物件写入顺序，并可附加编号文字点。

## 选项说明

- 输入：选择一条参考曲线；选择一个或多个物件；拾取一条方向线或参考线
- 输出：生成文字点；添加新的几何或标注结果；需要时把结果自动编组
- 可调项：按群组拆分、附加文字点、重映射
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择一条参考曲线、选择一个或多个物件、拾取一条方向线或参考线。
2. 根据需要调整命令行选项：按群组拆分、附加文字点、重映射。
3. 确认执行后，系统会生成文字点、添加新的几何或标注结果、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
