# B_CurveSnap

## 功能说明

将源曲线端点吸附到目标曲线附近的最近位置。

## 源码要点

- 输入：按界面提示完成输入。
- 输出：直接替换原始物件
- 可调项：吸附半径、仅最近端、节点吸附
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 根据需要调整命令行选项：吸附半径、仅最近端、节点吸附。
2. 确认执行后，系统会直接替换原始物件。

## 注意事项

- 命令会记住上一次使用的选项。
- 命令会直接替换源曲线端点位置，请先确认备份需求。
