# B_Planarization

## 功能说明

将单面 Brep 平面化，可预览包围盒、信息点和边角固定结果。

## 选项说明

- 输入：选择单面 Brep
- 输出：生成新的 Brep；直接替换原始物件；在命令行输出结果；需要时把结果自动编组
- 可调项：显示信息、显示包围盒、边角固定、边角固定选项、公差、删除输入、隐藏输入
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择单面 Brep。
2. 根据需要调整命令行选项：显示信息、显示包围盒、边角固定、边角固定选项、公差、删除输入、隐藏输入。
3. 确认执行后，系统会生成新的 Brep、直接替换原始物件、在命令行输出结果、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 支持删除输入物件时，执行前请确认是否需要保留原始数据。
- 支持隐藏输入物件时，便于和结果进行对比。
- 结果会受到模型公差或命令容差设置影响。
- 更适合单面 Brep 或单片曲面，结果受公差设置影响。
