# B_ImportLayerFromClipboard

## 功能说明

读取剪贴板中的图层结构，并在当前选中图层下创建对应子层。

## 源码要点

- 输入：先在图层面板选中目标图层
- 输出：在选中图层下创建导入的图层结构
- 可调项：无明显命令行选项。
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：先在图层面板选中目标图层。
2. 确认执行后，系统会在选中图层下创建导入的图层结构。

## 注意事项

- 依赖系统剪贴板，跨会话前请确认剪贴板内容没有被覆盖。
- 执行前需要先在图层面板选中目标图层。
- 执行前需要在图层面板选中一个或多个目标根图层。
