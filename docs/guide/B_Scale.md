# B_Scale

## 功能说明

以包围盒中心或指定原点为基准缩放物件。

## 选项说明

- 输入：先选择命令模式；选择一个或多个物件；按提示拾取点位
- 输出：直接变换原始物件
- 可调项：B Scale、By Center、By Origin
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：先选择命令模式、选择一个或多个物件、按提示拾取点位。
2. 根据需要调整命令行选项：B Scale、By Center、By Origin。
3. 确认执行后，系统会直接变换原始物件。

## 注意事项

- 命令会记住上一次使用的选项。
