# B_AddRadialDimension

## 功能说明

对圆、圆弧或圆形边缘生成径向或直径标注。

## 选项说明

- 输入：按命令过滤选择物件
- 输出：生成径向或直径标注；需要时把结果自动编组
- 可调项：尺寸类型、尺寸距离、尺寸位置、简化、二维模式、公差
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：按命令过滤选择物件。
2. 根据需要调整命令行选项：尺寸类型、尺寸距离、尺寸位置、简化、二维模式、公差。
3. 确认执行后，系统会生成径向或直径标注、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 结果会受到模型公差或命令容差设置影响。
