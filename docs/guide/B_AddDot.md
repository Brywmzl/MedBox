# B_AddDot

## 功能说明

按物件属性格式批量生成文字点，适合把属性值转成现场标记。

## 选项说明

- 输入：选择一个或多个物件
- 输出：生成文字点；需要时把结果自动编组
- 可调项：输出图层、复制属性
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择一个或多个物件。
2. 根据需要调整命令行选项：输出图层、复制属性。
3. 确认执行后，系统会生成文字点、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
