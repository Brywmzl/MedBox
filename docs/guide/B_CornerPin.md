# B_CornerPin

## 功能说明

以边角固定方式重建或调整曲面/多重曲面，并输出新的 Brep。

## 源码要点

- 输入：选择 Brep 物件
- 输出：生成新的 Brep；直接替换原始物件
- 可调项：合并全部边、删除输入、公差
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择 Brep 物件。
2. 根据需要调整命令行选项：合并全部边、删除输入、公差。
3. 确认执行后，系统会生成新的 Brep、直接替换原始物件。

## 注意事项

- 命令会记住上一次使用的选项。
- 支持删除输入物件时，执行前请确认是否需要保留原始数据。
- 结果会受到模型公差或命令容差设置影响。
