# B_ImportPoints

## 功能说明

打开点位导入对话框，解析文本或表格数据后生成点。

## 选项说明

- 输入：在导入点对话框中填写参数
- 输出：在对话框确认后执行对应处理
- 可调项：无明显命令行选项。
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 打开命令后，先在对话框中填写需要的参数或筛选条件。
2. 按提示准备输入：在导入点对话框中填写参数。
3. 确认执行后，系统会在对话框确认后执行对应处理。

## 注意事项

- 命令会记住上一次使用的选项。
