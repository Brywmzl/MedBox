# B_ObjectShare

## 功能说明

通过分享码把物件导出到共享目录，或按分享码导入。

## 源码要点

- 输入：选择一个或多个物件；输入关键字、替换文本或其他字符串参数
- 输出：导出分享文件并生成分享码，或按分享码导入共享物件
- 可调项：分享、获取、分享列表
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：选择一个或多个物件、输入关键字、替换文本或其他字符串参数。
2. 根据需要调整命令行选项：分享、获取、分享列表。
3. 确认执行后，系统会导出分享文件并生成分享码，或按分享码导入共享物件。

## 注意事项

- 依赖系统剪贴板，跨会话前请确认剪贴板内容没有被覆盖。
- 共享目录需要可访问，否则无法分享或读取。
