# B_Area

## 功能说明

统计所选闭合曲线、曲面、网格、SubD 或填充的面积。

## 源码要点

- 输入：按界面提示完成输入。
- 输出：在命令行输出结果
- 可调项：接受曲面、接受网格、接受闭合曲线、接受填充
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 根据需要调整命令行选项：接受曲面、接受网格、接受闭合曲线、接受填充。
2. 确认执行后，系统会在命令行输出结果。

## 注意事项

- 命令会记住上一次使用的选项。
- 该命令的主要结果通常显示在命令行，不一定会新增几何。
