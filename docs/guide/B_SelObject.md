# B_SelObject

## 功能说明

按同图层或选中图层批量选取物件，可包含子图层。

## 源码要点

- 输入：选择一个或多个物件；先在图层面板选中目标图层
- 输出：选中符合条件的物件
- 可调项：包含子图层、按同图层、按选中图层
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：选择一个或多个物件、先在图层面板选中目标图层。
2. 根据需要调整命令行选项：包含子图层、按同图层、按选中图层。
3. 确认执行后，系统会选中符合条件的物件。

## 注意事项

- 执行前需要先在图层面板选中目标图层。
- 只会处理当前可见范围内的内容。
