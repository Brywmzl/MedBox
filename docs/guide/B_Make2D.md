# B_Make2D

## 功能说明

按方向线或当前视图生成 2D 隐线或切剖线结果。

## 选项说明

- 输入：先选择命令模式；选择一个或多个物件；拾取一条方向线或参考线
- 输出：按命令逻辑处理当前输入
- 可调项：Direction、Viewport
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：先选择命令模式、选择一个或多个物件、拾取一条方向线或参考线。
2. 根据需要调整命令行选项：Direction、Viewport。
3. 确认执行后，系统会按命令逻辑处理当前输入。
