# B_Text

## 功能说明

对注释文字执行替换、清理、转纯文本、格式化、提取数字、复制等处理。

## 选项说明

- 输入：先选择命令模式；按命令过滤选择物件；输入关键字、替换文本或其他字符串参数
- 输出：直接修改文字内容，或把整理后的结果复制到剪贴板
- 可调项：无明显命令行选项。
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：先选择命令模式、按命令过滤选择物件、输入关键字、替换文本或其他字符串参数。
2. 确认执行后，系统会直接修改文字内容，或把整理后的结果复制到剪贴板。

## 注意事项

- 依赖系统剪贴板，跨会话前请确认剪贴板内容没有被覆盖。
- 部分处理模式会直接改写原文字内容。
