# B_ObjectNameReplace

## 功能说明

批量替换所选物件名称中的指定字符串。

## 源码要点

- 输入：选择一个或多个物件；输入关键字、替换文本或其他字符串参数
- 输出：按命令逻辑处理当前输入
- 可调项：无明显命令行选项。
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：选择一个或多个物件、输入关键字、替换文本或其他字符串参数。
2. 确认执行后，系统会按命令逻辑处理当前输入。
