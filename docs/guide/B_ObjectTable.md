# B_ObjectTable

## 功能说明

打开表格查看器，集中浏览所选物件的数据。

## 源码要点

- 输入：选择一个或多个物件；在表格查看器中填写参数
- 输出：打开表格查看器展示所选物件数据
- 可调项：无明显命令行选项。
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 打开命令后，先在对话框中填写需要的参数或筛选条件。
2. 按提示准备输入：选择一个或多个物件、在表格查看器中填写参数。
3. 确认执行后，系统会打开表格查看器展示所选物件数据。

## 注意事项

- 命令会记住上一次使用的选项。
