# B_PlanarHeightTest

## 功能说明

计算曲面偏离平面的最大弦高，并可上色或加文字点。

## 源码要点

- 输入：选择曲面或多重曲面
- 输出：生成文字点；需要时把结果自动编组
- 可调项：显示文字点、应用颜色、文字点位置、小数位
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择曲面或多重曲面。
2. 根据需要调整命令行选项：显示文字点、应用颜色、文字点位置、小数位。
3. 确认执行后，系统会生成文字点、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 结果受模型公差、面类型和点位取样方式影响。
