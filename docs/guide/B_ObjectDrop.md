# B_ObjectDrop

## 功能说明

将物件序列化到剪贴板，或从剪贴板还原物件与群组关系。

## 选项说明

- 输入：先选择命令模式；选择一个或多个物件
- 输出：把物件数据写入剪贴板，或从剪贴板恢复物件与群组
- 可调项：复制属性、复制群组、Copy、Paste
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：先选择命令模式、选择一个或多个物件。
2. 根据需要调整命令行选项：复制属性、复制群组、Copy、Paste。
3. 确认执行后，系统会把物件数据写入剪贴板，或从剪贴板恢复物件与群组。

## 注意事项

- 命令会记住上一次使用的选项。
- 依赖系统剪贴板，跨会话前请确认剪贴板内容没有被覆盖。
- 复制与粘贴都依赖系统剪贴板。
