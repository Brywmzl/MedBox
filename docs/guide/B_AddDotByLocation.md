# B_AddDotByLocation

## 功能说明

根据点坐标生成文字点，可设置基准平面与输出格式。

## 源码要点

- 输入：设置或拾取基准平面；按提示拾取点位
- 输出：添加新的几何或标注结果；在命令行输出结果；需要时把结果自动编组
- 可调项：手动、拾取、设置格式、设置基准、重置基准
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：设置或拾取基准平面、按提示拾取点位。
2. 根据需要调整命令行选项：手动、拾取、设置格式、设置基准、重置基准。
3. 确认执行后，系统会添加新的几何或标注结果、在命令行输出结果、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 该命令的主要结果通常显示在命令行，不一定会新增几何。
