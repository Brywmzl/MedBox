# B_ResetRtf

## 功能说明

清除注释物件的富文本内容，保留纯文本。

## 选项说明

- 输入：按界面提示完成输入。
- 输出：直接清除注释物件的富文本内容
- 可调项：无明显命令行选项。
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 确认执行后，系统会直接清除注释物件的富文本内容。

## 注意事项

- 只处理注释类物件，不会作用到普通几何。
