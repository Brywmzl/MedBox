# B_AddDotByLength

## 功能说明

对曲线或边缘的分段长度批量生成文字点标注。

## 源码要点

- 输入：按命令过滤选择物件
- 输出：生成文字点；需要时把结果自动编组
- 可调项：分段、位置、忽略短项、小数位、仅非直线边、线性容差
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：按命令过滤选择物件。
2. 根据需要调整命令行选项：分段、位置、忽略短项、小数位、仅非直线边、线性容差。
3. 确认执行后，系统会生成文字点、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 结果会受到模型公差或命令容差设置影响。
