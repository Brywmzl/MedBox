# B_CopyLayerToClipboard

## 功能说明

把选中图层的层级与颜色信息复制到剪贴板，供别处粘贴导入。

## 选项说明

- 输入：先在图层面板选中目标图层
- 输出：把图层结构写入剪贴板
- 可调项：无明显命令行选项。
- 选项记忆：本次源码未见持久化选项。

## 操作步骤

1. 按提示准备输入：先在图层面板选中目标图层。
2. 确认执行后，系统会把图层结构写入剪贴板。

## 注意事项

- 依赖系统剪贴板，跨会话前请确认剪贴板内容没有被覆盖。
- 执行前需要先在图层面板选中目标图层。
- 依赖系统剪贴板，适合与“导入图层”配合使用。
