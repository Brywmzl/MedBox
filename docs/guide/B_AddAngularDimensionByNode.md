# B_AddAngularDimensionByNode

## 功能说明

对曲线节点或折点自动生成角度标注，支持过滤直角与角度容差。

## 选项说明

- 输入：选择曲线或边缘
- 输出：生成角度标注；需要时把结果自动编组
- 可调项：半径、连接曲线、忽略直角、角度容差（度）
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择曲线或边缘。
2. 根据需要调整命令行选项：半径、连接曲线、忽略直角、角度容差（度）。
3. 确认执行后，系统会生成角度标注、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 结果会受到模型公差或命令容差设置影响。
