# B_AddLinearDimensionByNaked

## 功能说明

对 Brep 裸边批量生成尺寸标注。

## 选项说明

- 输入：选择 Brep 几何
- 输出：添加新的几何或标注结果
- 可调项：忽略扩展边、忽略短项、仅直线边、线性容差
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择 Brep 几何。
2. 根据需要调整命令行选项：忽略扩展边、忽略短项、仅直线边、线性容差。
3. 确认执行后，系统会添加新的几何或标注结果。

## 注意事项

- 命令会记住上一次使用的选项。
- 结果会受到模型公差或命令容差设置影响。
