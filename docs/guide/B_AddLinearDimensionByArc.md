# B_AddLinearDimensionByArc

## 功能说明

对弧线生成弧长或半径相关的尺寸标注。

## 源码要点

- 输入：选择曲线或边缘
- 输出：生成角度标注；需要时把结果自动编组
- 可调项：偏移、强制执行、显示弧长、显示半径、小数位、公差
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择曲线或边缘。
2. 根据需要调整命令行选项：偏移、强制执行、显示弧长、显示半径、小数位、公差。
3. 确认执行后，系统会生成角度标注、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 结果会受到模型公差或命令容差设置影响。
