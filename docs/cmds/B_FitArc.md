# B_FitArc

## 功能说明

对曲线尝试拟合圆弧，并可附带误差或半径文字点。

## 源码要点

- 输入：选择曲线物件
- 输出：生成文字点；生成圆弧；在命令行输出结果；需要时把结果自动编组
- 可调项：强制执行、显示文字点、小数位、设置格式、删除输入、公差
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择曲线物件。
2. 根据需要调整命令行选项：强制执行、显示文字点、小数位、设置格式、删除输入、公差。
3. 确认执行后，系统会生成文字点、生成圆弧、在命令行输出结果、需要时把结果自动编组。

## 注意事项

- 命令会记住上一次使用的选项。
- 支持删除输入物件时，执行前请确认是否需要保留原始数据。
- 结果会受到模型公差或命令容差设置影响。
