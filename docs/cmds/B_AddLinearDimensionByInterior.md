# B_AddLinearDimensionByInterior

## 功能说明

对 Brep 内角边缘批量生成尺寸标注。

## 源码要点

- 输入：选择 Brep 物件
- 输出：添加新的几何或标注结果
- 可调项：偏移、忽略短项、仅直线、二维模式
- 选项记忆：会记住上一次设置。

## 操作步骤

1. 按提示准备输入：选择 Brep 物件。
2. 根据需要调整命令行选项：偏移、忽略短项、仅直线、二维模式。
3. 确认执行后，系统会添加新的几何或标注结果。

## 注意事项

- 命令会记住上一次使用的选项。
