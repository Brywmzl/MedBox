# 外观约定

## 视觉方向

整体视觉以低饱和紫红系为主，气质应稳定、克制、专业，不做高对比霓虹风，也避免偏冷的科技蓝主导界面。

## 主配色

### 1. 品牌主色

- `rgb(104, 0, 75)`
- `#68004B`
- 建议变量名：`--color-brand-primary`
- 用途：主按钮、主标题强调、关键交互高亮、选中态、重点图形元素

### 2. 品牌辅助色

- `rgb(162, 100, 145)`
- `#A26491`
- 建议变量名：`--color-brand-secondary`
- 用途：次级按钮、标签、卡片描边、悬停态、渐变过渡色

### 3. 品牌浅底色

- `rgb(206, 175, 197)`
- `#CEAFC5`
- 建议变量名：`--color-brand-soft`
- 用途：浅色背景、信息区块底色、弱强调面板、输入区聚焦氛围色

## 使用原则

- 深色负责聚焦，中色负责过渡，浅色负责铺底。
- 页面大面积背景优先使用白色或接近白色的中性色，`--color-brand-soft` 只用于局部染色，不直接整屏铺满。
- 文字落在深色底上时使用白色或高亮浅色；文字落在浅底上时优先使用 `--color-brand-primary` 或深灰，不直接使用 `--color-brand-secondary` 作为正文色。
- 需要品牌渐变时，优先使用 `--color-brand-primary` 到 `--color-brand-secondary`，避免再混入第三个高饱和颜色。

## 推荐变量

```css
:root {
  --color-brand-primary: rgb(104, 0, 75);
  --color-brand-secondary: rgb(162, 100, 145);
  --color-brand-soft: rgb(206, 175, 197);

  --color-brand-primary-hex: #68004b;
  --color-brand-secondary-hex: #a26491;
  --color-brand-soft-hex: #ceafc5;
}
```

## 组件落位建议

- 按钮：主按钮用 `--color-brand-primary`，次按钮用 `--color-brand-secondary`。
- 卡片：默认浅底或白底，选中态可加 `--color-brand-soft` 背景和 `--color-brand-secondary` 描边。
- 导航与链接：当前项或重要链接用 `--color-brand-primary`，悬停时向 `--color-brand-secondary` 过渡。
- 图表与图标：优先围绕三色做层级，不额外引入同权重主色。
